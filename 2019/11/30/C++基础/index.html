②to_string、sto-x

to_string（）可以把int、float、double等类型变量转化为string型变量

stoi、stod可以将字符串string转化为对应的int型、double型变量

stoi如果遇到的是非法输入（比如stoi（"123.4"），123.4不是一个int型变量）：
1.会自动截取最前面的数字，直到遇到不是数字为止
（所以说如果是浮点型，会截取前面的整数部分）

2.如果最前面不是数字，会运行时发生错误

stod如果是非法输入：
1.会自动截取最前面的浮点数，直到遇到不满足浮点数为止
2.如果最前面不是数字或者小数点，会运行时发生错误
3.如果最前面是小数点，会自动转化后在前面补0

不仅有stoi、stod两种，相应的还有：

对星号*的总结

在我们目前所学到的语法中，星号*

  • 表示乘法，例如int a = 3, b = 5, c; c = a * b;，这是最容易理解的。
  • 表示定义一个指针变量，以和普通变量区分开，例如int a = 100; int *p = &a;。
  • 表示获取指针指向的数据，是一种间接操作，例如int a, b, *p = &a; *p = 100; b = *p;

①unordered_map/set的使用

unordered_map在头文件#include 中，unordered_set在头文件#include中

unordered_map/unordered_set和map/set，map/set会按照键值对的键key/元素进行排序，而unordered map/unordered set省去了这个排序的过程，但不是按照插入顺序排序，直接是无序的，如果偶尔刷题时候用map或者set超时了，可以考虑用这两者缩短代码运行时间、提高代码效率，用法和map、set是一样的。
